The PHP TD Gram PHP 7.4 FFI `libtdjson` Adapter


## Install

Via Composer

```bash
$ composer require phptdgram/adapter-ffi
```

## Testing

Run PHP style checker

```bash
$ composer cs-check
```

Run PHP style fixer

```bash
$ composer cs-fix
```

Run all continuous integration tests

```bash
$ composer ci-run
```
